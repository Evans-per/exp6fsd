# Expense Tracker

Simple React Expense Tracker (Vite) — add income/expenses, view totals, delete transactions.

Quick start

1. Install dependencies

```bash
cd expense-tracker
npm install
```

2. Run dev server

```bash
npm run dev
```

App features
- Add transactions (description, positive amount, type)
- Delete transactions
- Live totals: balance, income, expenses

Additional features implemented:
- Separate Income and Expenses sections
- Expense categories (Food, Transport, Utilities, Entertainment, Health, Other)
- Expense breakdown by category shown in the summary
- Background video (example external URL used). Replace `/bg.mp4` in project root or update the video URL in `src/App.jsx` to use your own video.
